#Angular 2 Seed

## Voraussetzungen

- node installiert
- npm installiert
- Angular Material CLI

Pakete installieren via 

> npm install

## Features

- Angular 2
- Typescript
- Webpack
- Karma 
- Jasmine
- Compodoc
- SCSS
- Compass
- Font Awesome
- Bootstrap 4 
- Angular Material

## Jobs

- Server / Compiler 

> npm run start

- Tests

> npm run test

- Build

> npm run build

- Docs

> npm run docs
